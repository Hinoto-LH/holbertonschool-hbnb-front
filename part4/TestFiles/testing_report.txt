## 1. User Endpoints

### POST /api/v1/users/ — Create User

#### ✅ TC-U-01 — Valid user creation
```bash
curl -X POST http://localhost:5000/api/v1/users/ \
  -H "Content-Type: application/json" \
  -d '{"first_name": "John", "last_name": "Doe", "email": "john.doe@example.com"}'
```
**Expected:** `201 Created`  
**Result:** ✅ PASS

---

| User     | first_name  | Non-vide,                                 |
| User     | last_name   | Non-vide,                                 |
| User     | email       | Format valide `x@x.xx`                    |
| Place    | title       | Non-vide, max 100 chars                   |
| Place    | price       | Nombre > 0                                |
| Place    | latitude    | Float entre -90 et 90                     |
| Place    | longitude   | Float entre -180 et 180                   |
| Place    | owner_id    | Non-vide                                  |
| Review   | text        | Non-vide                                  |
| Review   | rating      | Entier entre 1 et 5                       |
| Review   | user_id     | Non-vide                                  |
| Review   | place_id    | Non-vide                                  |
| Amenity  | name        | Non-vide,                                 |
